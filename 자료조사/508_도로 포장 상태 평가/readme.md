#### 종단평탄성
도로의 종단면 평탄도
#### 소성변형
차량 통행으로 인해 발생하는 포장면의 영구적인 변형
#### 균열
도로 표면에 발생하는 균열의 형태와 범위<br>
<br>
SPI = 10 - PDI<br>
PDI = ((10-SPI1)**5 + (10-SPI2)**5 + (10-SPI3)**5)**0.2<br>
SPI1 = 10 - 2.23 * (포장의 균열율)**0.3<br>
SPI2 = 10 - 0.2 * 평균 소성변형 깊이<br>
SPI3 = 10 - 0.531 * 종단 평탄성<br>
